# 📋 App de Tarefas - RecyclerView

Este projeto é um **aplicativo para registro de tarefas**, desenvolvido durante a aula de **Desenvolvimento Mobile** na **Universidade Santa Cecília (Unisanta)**.
O aplicativo permite adicionar tarefas com **nome e descrição**, exibindo os itens em uma lista usando **RecyclerView**, com status automático de **pendente** e opção para marcar como **concluída**.

## ⚙️ Funcionalidades

* ➕ **Adicionar tarefas:** o usuário pode inserir nome e descrição da tarefa.
* 📋 **Lista com RecyclerView:** exibição dinâmica das tarefas cadastradas.
* ⏳ **Status automático:** toda tarefa é criada como *pendente*.
* ✅ **Marcar como concluída:** botão em cada item da lista para finalizar a tarefa.
* 🔄 **Atualização em tempo real:** a lista é atualizada automaticamente ao adicionar ou concluir tarefas.
* 📱 **Interface simples e funcional:** layout desenvolvido em XML seguindo o padrão Android.

---

## 🧠 Tecnologias Utilizadas

* **Kotlin** – linguagem principal de desenvolvimento
* **Android Studio** – ambiente de desenvolvimento (IDE)
* **RecyclerView** – exibição de lista dinâmica
* **XML (Android Layout)** – criação da interface gráfica
* **Adapter / ViewHolder** – controle dos itens da lista

---

## 🧩 Estrutura do Aplicativo

O aplicativo possui:

* Tela principal com lista de tarefas
* Campo para inserir nome da tarefa
* Campo para inserir descrição
* Botão para adicionar tarefa
* Botão para marcar como concluída dentro do RecyclerView

Cada tarefa possui:

* Nome
* Descrição
* Status (Pendente / Concluída)

## ▶️ Como Executar o Projeto

### 🔧 Requisitos

* Android Studio instalado
* SDK Android configurado

### ▶️ Passos

1. Clone o repositório:

```bash
git clone https://github.com/Borgesslf/TarefasApp.git
```

2. Abra o projeto no Android Studio
3. Aguarde o Gradle sincronizar
4. Execute em um emulador ou celular Android
